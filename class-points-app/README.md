# 班级积分管理系统

教师端轻量化本地应用：学生积分数字化管理 + 课堂趣味辅助工具，支持**语音交互**，无网络即可运行（语音识别需 Chrome/Edge 在线）。

## 功能概览

| 模块 | 功能 |
|------|------|
| **学生管理** | Excel 批量导入/导出（姓名、学号、班级）；单/批量增删改；姓名/学号模糊搜索 |
| **积分操作** | 单/批量加减分，预设规则，操作时间与原因自动记录（记录不可删）；**语音指令**解析后自动填表，可一键确认或手动修正 |
| **数据统计** | 学生积分曲线（日/周/月）、班级排名、奖惩类型分布、语音 vs 手动占比；支持导出报表 |
| **教学工具** | 随机点名（可排除已点、语音播报、记录频次）；养宠互动（积分兑换宠物/道具、排行榜）；成语接龙（内置词库+自定义、接龙正确可加分） |

## 技术说明

- **本地运行**：数据存于浏览器 IndexedDB，不依赖后端与网络（除语音识别需调用浏览器在线能力）。
- **快捷键**：`F1` 跳转点名，`F2` 跳转并唤醒语音。
- **语音**：使用浏览器 Web Speech API（建议 Chrome/Edge），支持普通话；指令不完整时会提示补全。

## 快速开始

### 网页版（推荐先试用）

```bash
cd class-points-app
npm install
npm run dev
```

浏览器打开 http://localhost:5173 ，用 Chrome 或 Edge 以获得语音支持。

### 桌面版（Electron）

```bash
npm install
npm run build
npm run electron
```

开发时可同时起前端与 Electron：

```bash
npm run electron:dev
```

打包安装包：

```bash
npm run electron:build
```

## 使用提示

1. **首次使用**：在学生管理页导入 Excel（需含「姓名」「学号」列，可选「班级」），或手动添加学生。
2. **语音加分**：在积分操作页点击「语音操作」或按 F2，说出如“给张三加2分 主动答题”“全班减1分 纪律差”，识别结果会填入表单，确认或修改后点「确认提交」。
3. **点名**：教学工具 → 随机点名，可勾选排除已点学生，点击「点名」随机抽取并语音播报。

## 隐私与兼容

- 麦克风仅在点击「语音操作」时请求，用完即停。
- 数据仅存本地，不上传。
- 建议使用 Chrome 或 Edge 以获得最佳兼容与语音识别效果。
